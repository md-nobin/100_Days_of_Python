Converting Data Types
We can convert a variable from 1 data
type to another.
Converting to float:
float()
Converting to int:
int()
Converting to string:
str()

Checking Data Types
We can use the type() function
to check what is the data type of a
particular variable.

n = 3.14159
type(n) #result float
